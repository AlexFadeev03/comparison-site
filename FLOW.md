# User Flow & Role Scenarios

## 1. Гість (незареєстрований користувач)
- Перегляд головної сторінки з брендингом, категоріями, топ-10 продуктів.
- Перегляд списку/гріду продуктів з фільтрацією за категоріями.
- Перегляд детальної інформації про продукт.
- Додавання продуктів до порівняння (через localStorage/cookie, не зберігається на сервері).
- Сторінка порівняння: side-by-side таблиця для 2-3 продуктів, кнопки "видалити"/"очистити всі".
- Не може додавати/редагувати продукти, залишати відгуки/рейтинги, додавати у wishlist.

## 2. Зареєстрований користувач
- Всі можливості гостя.
- (Опціонально) Може зберігати порівняння у профілі, додавати продукти у wishlist, залишати відгуки/рейтинги.
- Може редагувати/видаляти свої відгуки.

## 3. Адміністратор
- Всі можливості користувача.
- Доступ до адмін-панелі (через окремий маршрут, middleware).
- CRUD для продуктів: додавання, редагування, видалення.
- (Опціонально) CRUD для категорій/підкатегорій.
- (Опціонально) Може встановлювати порядок для Top 10 продуктів.
- (Опціонально) Керування користувачами (ролі, бан).

---

## Основний флоу для всіх ролей
1. **Головна сторінка:**
   - Брендинг, категорії, топ-10 продуктів.
2. **Сторінка продуктів:**
   - Фільтрація, сортування, адаптивний дизайн.
3. **Сторінка порівняння:**
   - Таблиця з Name, Image, Price, Rating, Pros, Cons, Features. Видалення/очищення.
4. **Адмін-панель:**
   - CRUD для продуктів (і, опціонально, для категорій/підкатегорій).
